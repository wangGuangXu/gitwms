# 吉特仓储管系统基础版本
  适合单仓库，基本的仓库入库管理，出库管理，盘点，报损，移库，库位等管理，有着可视化图表。 系统采用Bootstrap 开发，UI 相对比较简单，业务功能不复杂，适合一般的学习开发者。
